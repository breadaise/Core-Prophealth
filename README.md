If you think props are overpowered, Core's Prophealth can easily fix that. All the props now has a health bar that can be reinforced with in-game money!

This addon made and designed for DarkRP, no promises if it's gonna work on any other gamemode. 

**Featuring:**

- Players can increase their props health with just a click of a button!
- Drag n' Drop; No configuration or additional content needed.
- Customizable; Edit everything in-game through the UI.
- Clean & modern UI; No framework/library is required.
- Lightweight; Optimized to work on a large player base servers!

**Compatibility:**

- ULX
- DarkRP 2.7+
